# RefWatch

RefWatch is a simple and intuitive timer app for sports referees, built for Wear OS devices. It helps you keep track of the game time with ease.

## Main Features

- **Match Timer:** A clear and large display of the match timer. You can easily start, pause, and reset the timer.
- **Halftime Management:** The app supports multiple periods (e.g., halftimes in soccer). You can switch between periods.
- **Time Tracking:**
    - **Elapsed Time:** Shows the current running time of the period.
    - **Remaining Time:** Shows how much time is left in the period.
    - **Additional Time:** A separate display for stoppage or extra time.
- **Game Settings:** You can configure the duration of the periods to fit the rules of your game.
- **Haptic Feedback:** The app provides haptic feedback for important actions like starting or pausing the timer, so you can be sure your action was registered without looking at the watch.
- **Designed for Wear OS:** The user interface is optimized for smartwatches, with large touch targets and clear information display.

## How to Use

1.  **Start the game:** Open the app and tap the play button to start the timer.
2.  **Pause the game:** Tap the pause button to stop the timer.
3.  **Reset:** After a period or game, you can reset the timer.
4.  **Switch Halftime:** Tap on the main clock to switch to the next period.
5.  **Settings:** Before starting, go to settings to adjust the length of the periods.
