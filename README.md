# RefWatch

A simple and effective Wear OS application for soccer referees to keep track of match time, including regulation and stoppage time.

## Features

*   **Match Timer:** A primary timer for each half of the game.
*   **Additional Time:** Automatically starts a count-up timer for additional time after the regulation time has expired.
*   **Period Switching:** Easily switch between the first and second halves.
*   **Configurable Game Settings:** (Planned) Adjust timer settings for different game formats.
*   **Simple Controls:** Start, stop, and reset the timer with ease.

## Getting Started

This project is a standard Android application built with Kotlin and Jetpack Compose for Wear OS.

### Building

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Build the project using Gradle: `./gradlew assembleDebug`
4.  Deploy to a Wear OS emulator or physical device.
